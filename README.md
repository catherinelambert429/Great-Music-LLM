# Great Music LLM - Landing Page

An elegant, sophisticated landing page for Great Music LLM, a high-end orchestra-focused business featuring world-class musicians.

## Project Overview

This React-based landing page showcases Great Music LLM's premium orchestral services, featuring:

- **Elegant Design**: Sophisticated color palette with deep navy, rich gold, and burgundy accents
- **Responsive Layout**: Fully responsive design optimized for all devices
- **Smooth Animations**: Elegant transitions and animations using Framer Motion
- **Professional Content**: Sections highlighting services, musicians, and contact information

## Features

- **Hero Section**: Eye-catching introduction with call-to-action buttons
- **About Section**: Company overview with key statistics
- **Services Section**: Display of event types (Weddings, Corporate Events, Private Concerts, Galas)
- **Musicians Section**: Featured artists showcase with credentials
- **Contact Section**: Contact form and contact information
- **Responsive Navigation**: Smooth scroll navigation with mobile menu

## Tech Stack

- **React** - UI library
- **Vite** - Build tool and dev server
- **Framer Motion** - Animation library
- **React Icons** - Icon library
- **React Scroll** - Smooth scrolling functionality

## Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd Great-Music-LLM
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open your browser and navigate to the local development URL (typically `http://localhost:5173`)

## Build for Production

```bash
npm run build
```

The production-ready files will be in the `dist` folder.
